# Workflow: Online Course Creation

## Objective
Build complete, sellable online courses for clients in any industry.

## Required Inputs
- Client name and area of expertise
- Target student (who they're teaching, skill level)
- Course outcome (what students will be able to do after completing)
- Preferred platform (Teachable, Kajabi, Thinkific, Gumroad, self-hosted)
- Format preference (video scripts, written lessons, worksheets, quizzes, or mix)
- Course length target (mini course 1-3h / standard 5-10h / flagship 10h+)
- Price point and positioning (budget, mid, premium)

## Agent Roles
- **Queen (Orchestrator)**: Oversees full course build, ensures learning outcomes are met
- **Curriculum Designer**: Course outline, module structure, learning objectives per lesson
- **Researcher**: Industry knowledge gaps, competitor courses, pricing benchmarks
- **Scriptwriter**: Full video scripts or written lesson content per module
- **Worksheet/Resource Creator**: Workbooks, templates, checklists, quizzes
- **Sales Page Copywriter**: Course sales page, email launch sequence, checkout copy
- **Tech Setup Agent**: Platform configuration, payment setup, delivery automation

## Process
1. Queen receives brief → validates course viability (is there demand?)
2. Researcher validates topic, analyzes competitor courses, confirms pricing
3. Curriculum Designer builds full course outline (modules + lessons)
4. Queen approves outline → Scriptwriter and Resource Creator work in parallel
5. Sales Page Copywriter writes sales page + 5-email launch sequence
6. Tech Setup Agent configures platform
7. Queen does final review → delivers package

## Output Format
- Full course outline with learning objectives
- All lesson scripts or written content
- All worksheets, templates, and resources
- Sales page copy + email sequence
- Platform setup instructions
- Delivered to: `deliverables/courses/[client-name]/`

## Edge Cases
- If client has no audience → recommend lead magnet + waitlist strategy before launch
- Certification courses (coaching, fitness, finance) may need legal disclaimer review
- If client wants video production → scope separately, deliver scripts only in this workflow
